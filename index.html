Esercitazione interattiva per la classe 5ª - Prerequisito Web API

Metodo GET

Utilizzato per leggere dati. È un metodo sicuro e idempotente: chiamarlo 1 o 100 volte non cambia lo stato del server.

Scenario Didattico:

"Stai chiedendo al server di mostrarti l'elenco dei post nel database."

curl https://jsonplaceholder.typicode.com/posts/1

Cosa osservare:

  • Il server risponde con un oggetto JSON.
  • Lo Status Code dovrebbe essere 200 OK.

Mapping .NET:

[HttpGet]

Creato per scopi didattici - Utilizza l'API pubblica JSONPlaceholder.
